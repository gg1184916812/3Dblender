# Smart Align Pro CAD 功能測試指南

## 🧪 測試準備

### 1. 安裝模組化版本
1. 在 Blender 中，進入 `編輯 > 偏好設定 > 插件`
2. 啟用 "Smart Align Pro（繁體中文版 - 模組化）"
3. 禁用原版本以避免衝突
4. 重啟 Blender

### 2. 創建測試場景
```
- 創建 3-4 個不同形狀的 Cube
- 創建 1-2 個 Sphere
- 創建 1 個 Plane 作為地面
- 隨機移動和旋轉這些物件
```

## 🎯 CAD 功能測試

### 測試 1: CAD 吸附模式

#### 步驟：
1. **選擇目標物件**: 選擇一個 Cube 作為 Active Object
2. **選擇來源物件**: Shift+選擇其他物件
3. **啟動 CAD 模式**: 在面板中點擊 "CAD 吸附"
4. **選擇 From 點**: 
   - 移動鼠標到目標物件的頂點附近
   - 觀察是否顯示頂點吸附提示
   - 左鍵確認
5. **選擇 To 點**:
   - 移動鼠標到來源物件的邊緣附近
   - 觀察邊緣吸附提示
   - 左鍵確認
6. **執行對齊**: 按 Enter

#### 預期結果：
- From 點和 To 點應該精確吸附到幾何特徵
- 對齊應該基於選擇的點位進行
- 物件應該正確對齊

### 測試 2: 約束軸功能

#### 步驟：
1. 啟動 CAD 吸附模式
2. 選擇 From 點
3. **按 Tab** 切換約束軸：
   - 第一次：X 軸約束
   - 第二次：Y 軸約束  
   - 第三次：Z 軸約束
   - 第四次：無約束
4. 觀察鼠標移動時的約束效果
5. 選擇 To 點並執行對齊

#### 預期結果：
- 在約束軸模式下，鼠標移動應該只沿約束軸方向
- 對齊應該遵循約束軸限制

### 測試 3: 多種吸附模式

#### 測試不同吸附類型：
1. **頂點吸附**: 移動到物件頂點
2. **邊緣吸附**: 移動到物件邊緣
3. **面吸附**: 移動到物件表面
4. **中點吸附**: 移動到邊緣中點
5. **中心吸附**: 移動到物件中心

#### 使用快速吸附：
1. 點擊 "快速吸附" 按鈕旁的箭頭
2. 選擇特定吸附類型
3. 觀察是否只吸附到指定類型

### 測試 4: CAD 對齊類型

#### 測試不同對齊方式：
1. **兩點對齊**: 基本點對點對齊
2. **三點對齊**: 平面對齊（需要三個點）
3. **表面法線**: 法線對齊
4. **平行對齊**: 平行於 From-To 向量
5. **垂直對齊**: 垂直於 From-To 向量

## 🔧 核心功能測試

#### 測試快捷鍵：
1. **Alt+1**: 兩點對齊
2. **Alt+2**: 三點對齊

1. 選擇目標和來源物件
2. 按對應快捷鍵

## 🐛 故障排除

### 常見問題及解決方案：

#### 1. 模組導入錯誤
**問題**: `ImportError: cannot import name 'cad_snap'`
**解決**: 確保使用 `smart_align_pro_modular.py` 作為插件入口

#### 2. CAD 吸附不工作
**問題**: 吸附點檢測失敗
**解決**: 
- 檢查吸附容差設置
- 確認物件類型為 Mesh
- 檢查物件是否可見

#### 3. Modal 模式卡住
**問題**: 無法退出 CAD 模式
**解決**: 按 ESC 強制退出

#### 4. 預覽材質問題
**問題**: 預覽材質不正確
**解決**: 清除預覽並重新創建

#### 5. 快捷鍵衝突
**問題**: 快捷鍵無響應
**解決**: 檢查是否有其他插件使用相同快捷鍵
