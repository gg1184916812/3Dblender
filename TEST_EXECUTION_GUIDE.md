# Smart Align Pro 測試執行指南

## 🎯 測試目標：驗證 2.1.0 版本的真實可用性

### **測試原則**
- 誠實記錄，不美化結果
- 重現問題，不隱藏缺陷
- 專注核心，不追求完美
- 實用導向，不過度技術

---

## 🚀 快速測試 (5分鐘)

### **準備工作**
1. **安裝插件**: 在 Blender 中安裝 Smart Align Pro 2.1.0
2. **重啟 Blender**: 確保插件正確載入
3. **打開控制台**: 確保能看到錯誤訊息

### **測試 1: 基礎安裝檢查**
1. **檢查面板**: 側邊欄 → 對齊 → 是否看到面板？
2. **檢查按鈕**: 核心 4 功能按鈕是否顯示？
3. **檢查設置**: 設置面板是否正常？

**結果記錄**:
- ✅ 面板正常顯示
- ✅ 按鈕可以點擊
- ✅ 設置可以調整

---

## 🧪 核心功能測試 (15分鐘)

### **測試 2: 兩點對齊 (Alt+1)**

#### **準備場景**:
1. 創建一個大 Cube (2m)
2. 創建一個小 Cube (1m)
3. 將小 Cube 放置在大 Cube 附近

#### **執行測試**:
1. 選擇大 Cube (按 Shift 選擇小 Cube)
2. 按 Alt+1 或點擊 "兩點對齊" 按鈕
3. 觀察對齊結果

#### **記錄結果**:
```
操作步數: [數字] 步
執行時間: [時間] 秒
對齊效果: 成功/部分成功/失敗
是否需手動微調: 是/否
錯誤訊息: [如有]
```

### **測試 3: 預覽系統 (Alt+S/D/F)**

#### **準備場景**:
1. 使用上面的測試場景
2. 確保兩個 Cube 都被選中

#### **執行測試**:
1. 按 Alt+S (創建預覽)
2. 觀察是否出現半透明藍色預覽
3. 按 Alt+D (應用預覽)
4. 按 Alt+F (清除預覽)

#### **記錄結果**:
```
預覽創建: 成功/失敗
預覽顯示: 清晰/模糊/不可見
預覽應用: 成功/失敗
預覽清除: 成功/失敗
```

### **測試 4: 三點對齊 (Alt+2)**

#### **準備場景**:
1. 創建一個大 Cube (2m)
2. 創建一個小 Cube (1m)
3. 將小 Cube 旋轉一定角度

#### **執行測試**:
1. 選擇大 Cube (按 Shift 選擇小 Cube)
2. 按 Alt+2 或點擊 "三點對齊" 按鈕
3. 觀察對齊結果

#### **記錄結果**:
```
操作步數: [數字] 步
執行時間: [時間] 秒
平面對齊: 成功/部分成功/失敗
翻面選項: 正常/異常
是否需手動微調: 是/否
```

## 🎯 CAD 功能測試 (10分鐘)

### **測試 5: CAD 吸附模式**

#### **準備場景**:
1. 創建一個複雜的網格物件
2. 創建一個簡單的 Cube
3. 將它們放置在附近

#### **執行測試**:
1. 選擇兩個物件
2. 點擊 "CAD 吸附" 按鈕
3. 嘗試選擇 From 點 (左鍵點擊)
4. 嘗試選擇 To 點 (左鍵點擊)
5. 按 Enter 執行對齊

#### **記錄結果**:
```
模式啟動: 成功/失敗
From 點選擇: 順暢/卡頓/失敗
To 點選擇: 順暢/卡頓/失敗
吸附檢測: 準確/不準確
Hover Preview: 有/無/卡頓
約束切換: 正常/異常
```
